Structure du projet

Architecture globale

Le projet est organisé selon une architecture MVC (Modèle-Vue-Contrôleur) simplifiée :

docker-sae203/
│   dockerfile
│   README.md
│   
└───siteweb
    │   account.html
    │   index.html                                                                    
    │   login.html
    │   video.html
    │   videos.json
    │   
    ├───images
    ├───js
    ├───style
    └───videos

Conteneur Docker

Le projet est conteneurisé avec Docker pour une installation et un déploiement simplifiés.

Commandes principales

docker build -t ton_tube .

Construit l'image Docker

docker run -d -p 8080:80 ton_tube

Lance le conteneur

Structure des fichiers

Pages HTML

  • index.html - Page d'accueil avec liste des vidéos
  • video.html - Page de visionnage d'une vidéo
  • account.html - Page de gestion du compte utilisateur
  • login.html - Page de connexion

Scripts JavaScript

  • auth.js - Gestion de l'authentification
  • video.js - Gestion des vidéos
  • commentaires.js - Système de commentaires
  • recherche.js - Fonctionnalité de recherche
